# Supermarket OOP System (C++)

A console-based supermarket management system built using Object-Oriented Programming principles in C++.

---

## 📌 Features

- Add products (Normal, Weighted, Discounted)
- Remove products by ID
- Update product quantity
- Prevent duplicate product IDs
- Calculate grand total dynamically
- Exception handling for invalid input
- Memory-safe destructor cleanup

---

## 🧠 OOP Concepts Used

- Inheritance
- Polymorphism
- Virtual functions
- Encapsulation
- Dynamic memory management
- Exception handling

---

## 🛠 Technologies

- C++
- STL (`vector`)
- Object-Oriented Design

---

## ▶️ How to Compile & Run

```bash
g++ main.cpp -o market
./market
```

---

## 📂 Project Structure

- `main.cpp` – Contains product classes, shopping cart logic, and interactive menu system.

---

## 🎯 Project Purpose

This project demonstrates practical implementation of core OOP concepts through a fully interactive console application.

It simulates a basic supermarket system with multiple product types and dynamic behavior.
